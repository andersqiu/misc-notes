# Introduction
Dependent packages for building Python 3.6 from source code

## Environments:
* CentOS 7.4 (with gcc 4.8.5)

## Dependent packages for building Python 3.6 from source code
* bzip2-devel
* ncurses-devel
* gdbm-devel
* xz-devel
* sqlite-devel
* openssl-devel
* tk-devel
* readline-devel

## Command line:

```
$ sudo yum install -y bzip2-devel ncurses-devel gdbm-devel xz-devel sqlite-devel openssl-devel tk-devel readline-devel
```
